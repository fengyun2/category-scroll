# category-scroll

> Vue 实现左右菜单联动实现,类似于京东 tab 栏中的分类联动效果

## 启动

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Lints and fixes files

```
yarn run lint
```
